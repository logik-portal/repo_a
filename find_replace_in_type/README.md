# Find Replace In Type

**Script Version:** 1.0.0  
**Flame Version:** 2026  
**Written by:** Kieran Hanrahan  
**Creation Date:** 03.02.25  
**Update Date:** 04.21.25  

## Description

This script will find a specified search string within a Type TimelineFX and
replace that search term with something else without having to enter the Type
editor.
<br><br>
Works on segments or sequences containing Type TimelineFX.  For sequences, it will
find all segments that have Type TimelineFX and perform the find & replace.

## URL

http://github.com/khanrahan/find-replace-in-type

## Menus

- Right-click selected segments in a sequence → Edit... → Find and Replace in Type TimelineFX
- Right-click selected sequence or sequences → Edit... → Find and Replace in Type TimelineFX

## Installation

For all users, copy this file to:
/opt/Autodesk/shared/python
<br><br>
For a specific user, copy this file to:
/opt/Autodesk/user/<user name>/python
